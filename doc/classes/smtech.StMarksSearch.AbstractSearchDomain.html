__construct()

__construct(array<mixed,mixed>  $params)

Construct a SearchDomain

Parameters are passed to this constuctor as an associative array of values, including, at a minimum:

[
  'name' => 'A human-readable name for the domain',
  'url' => 'URL of the home page of the domain'
]

Subclasses may (and will) add additional required parameters to that list.

Parameters

array<mixed,mixed> $params

Throws

\Exception

If $params['url'] is not a valid URL or $params['name'] is empty.
